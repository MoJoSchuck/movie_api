Welcome to MyFlix API Documentation

This API provides access to information about movies, directors, and genres. Users can interact with the API to retrieve data, update their profiles, and manage their list of favorite movies.

MyFlix API Endpoints

Method Type Endpoint URL Description Request Format Response Format
GET /movies Returns a list of all movies. N/A JSON
GET /movies/:title Returns data about a single movie based on the title. N/A JSON
GET /movies/genre/:genreName Returns data about a genre based on the name. N/A JSON
GET /movies/directors/:directorName Returns data about a director based on the name. N/A JSON
POST /users Allows a new user to register. JSON Text message (success or error message)
PUT /users/:id Allows users to update their user information. User data (JSON) Text message (success or error message)
POST /users/:id/:movieTitle Allows users to add a movie to their favorites list. N/A Text message (success or error message)
DELETE /users/:id/:movieTitle Allows users to remove a movie from their favorites list. N/A Text message (success or error message)
DELETE /users/:id Allows existing users to log out. N/A Text message (success or error message)
